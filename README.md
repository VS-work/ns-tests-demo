# An example of NativeScript Application that contains proper Madium Tests


## How to prepare

First of all, you should install NativeScript CLI globally and run `npm i`.

## How to run the application

### Android

`tns run android`

### Debug under Android

`tns debug android`

### iOS

`tns run ios`

### Debug under iOS

`tns debug ios`

## How to run tests

It's quite easy: `npm test`
